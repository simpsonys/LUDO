feat: 세션 데이터 접근을 위한 MCP 도구 레이어 구현

- 사용자의 요청에 따라 LUDO의 세션 데이터에 구조적으로 접근할 수 있는 MCP(Mission Control Platform) 호환 도구 레이어의 첫 번째 버전을 구현했습니다.
- 별도의 서버 없이 Tauri 내부에서 동작하는 가벼운 In-process 방식으로 구현하여 기존 아키텍처 변경을 최소화했습니다.
- `@ludo/mcp-types` 패키지를 새로 추가하여 도구의 입출력 JSON 계약을 명확히 정의했습니다.
- Rust 백엔드에 `mcp.rs` 모듈을 추가하고, 단일 진입점인 `mcp_tool_dispatch` 명령어를 통해 다음 3가지 도구를 노출합니다:
  1. `get_session_summary`: 세션 메타데이터와 아티팩트 목록 요약
  2. `search_transcript`: 세션 트랜스크립트(`events.jsonl`) 내용 검색
  3. `read_artifact`: 지정된 아티팩트 파일 내용 읽기
- 기능 검증을 위해 앱 하단에 간단한 MCP 도구 테스트용 UI를 추가했습니다.
